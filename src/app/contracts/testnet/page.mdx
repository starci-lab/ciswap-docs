# 📦 CiSwap Smart Contracts

The following contracts are deployed on the **Aptos Testnet**. These modules power the core logic behind **CiSwap’s Virtual AMM** and **FA Factory**.

---

## 🧮 Swap Module

The **Swap Module** is the heart of CiSwap’s virtual liquidity system. It handles:

- Swap execution using the modified constant product formula:  
  \[(X + ciX)(Y + ciY) = k\]
- Pool creation and ciToken management
- Internal accounting of synthetic reserves
