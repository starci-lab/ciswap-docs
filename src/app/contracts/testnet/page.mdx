# 📦 CiSwap Smart Contracts

The following contracts are deployed on the **Aptos Testnet**. These modules power the core logic behind **CiSwap’s Virtual AMM** and **ciToken Factory**.

---

## 🧮 Swap Module

The **Swap Module** is the heart of CiSwap’s virtual liquidity system. It handles:

- Swap execution using the modified constant product formula:  
  \[(X + ciX)(Y + ciY) = k\]
- Pool creation and ciToken management
- Internal accounting of synthetic reserves

## 🏗️ FA Factory Module

The **FA (Fungible Asset) Factory Module** handles minting of internal debt tokens (`ciTokenX`, `ciTokenY`) that simulate liquidity. It is used to:

- Deploy new ciTokens on demand  
- Assign metadata and permissions  
- Plug ciTokens into other protocols (staking, lending, synthetic assets, etc.)
